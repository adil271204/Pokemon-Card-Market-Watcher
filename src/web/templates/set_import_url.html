Hinweise zur URL-Kompatibilität
  • Das System erkennt Karten anhand von Nummern wie 001/165 im Seitentext.
  • Seiten mit strukturierten Tabellen oder Listen werden am besten erkannt.
  • JavaScript-gerenderte Seiten (SPA) können nicht gelesen werden.
  • Kein eBay-Scraping – diese Funktion betrifft nur externe Kartenlisten-Seiten.
  • Wenn keine Karten erkannt werden: CSV-Import ist immer verfügbar.
